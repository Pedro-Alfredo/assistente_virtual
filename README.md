# Assistente Virtual (Modo Demo)

Projeto de assistente virtual em Python + Flask, com interface web.

## Funcionalidades
- Recebe mensagens simuladas de WhatsApp.
- Responde automaticamente com IA leve em português.
- Interface web para monitorar conversas.
- Modo demo, sem integração real com WhatsApp.

## Como usar
1. Instalar dependências:
   pip install -r requirements.txt
2. Executar servidor:
   python app.py
3. Abrir navegador: http://127.0.0.1:5000
